# Smart Assistive System for Alzheimer's Patients Using IoT and Machine Learning

# Overview
This project is an IoT-based healthcare monitoring system designed to improve the safety and quality of life of Alzheimer's patients. The system continuously monitors vital signs, tracks patient location, detects falls, and provides real-time alerts to caregivers.

# Features
- Real-time health monitoring
- Heart Rate and SpO₂ monitoring
- ECG and Temperature monitoring
- Fall detection using sensors
- GPS-based location tracking
- Facial recognition using Machine Learning
- Emergency SOS alerts
- Cloud-based dashboard using Firebase
- Caregiver notification system

# Technologies Used
- ESP8266 / ESP32
- Arduino IDE
- Python
- OpenCV
- Firebase
- GPS & GSM Modules
- IoT Sensors
- Machine Learning
- React.js

# Applications
- Home healthcare monitoring
- Alzheimer's patient tracking
- Elderly care systems
- Smart healthcare solutions
